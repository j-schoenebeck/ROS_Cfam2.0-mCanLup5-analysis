# Repository for Labrador retriever genome assembly ROS_Cfam2.0_v5.3 | mCanLup5
## Background
This repository provides supporting data and information for the Verkko assembly of "Alfie", a male yellow Labrador retriever dog. Raw data was generated by the Schoenebeck (University of Edinburgh) and Phillippy group (NHGRI). Annotation data was produced in collaboration with the Ostrander group (NHGRI) with the support of a NISC flagship support awarded to the aforementioned groups.
